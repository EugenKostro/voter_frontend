### Funkcionalnosti aplikacije
Voter sadrži sljedeće funckionalnosti: 
- Početna stranica
- Prijava i registracija
- Kreiranje sobe za anonimno glasanje
- Generiranje poveznice za slanje drugim glasačima
- Pregled statistike
